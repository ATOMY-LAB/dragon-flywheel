# ePaddle · Dragon Boat Data Flywheel · 龙舟数据飞轮

One-page interactive reference companion to **`ePaddle 202607.pptx`** (pitch deck, July 2026).

## Usage

Open [index.html](index.html) in any modern browser (double-click works — it is fully
self-contained: no network, no external fonts/icons/scripts). Bilingual: 中文 by default
with an EN toggle in the header (choice remembered via localStorage). Light and dark mode
follow the OS setting. Printing hides the interactive buttons and language toggle.

- **Ring buttons / nodes** isolate one of the six growth loops on the flywheel diagram and
  show its detail panel (stakeholders, ePaddle specifics, NBA/SailGP parallels).
- **Hard questions** is a Socratic investor Q&A (July 2026) — Don Valentine-style
  pressure-test questions with the founder's committed answers, usable as pitch prep.
